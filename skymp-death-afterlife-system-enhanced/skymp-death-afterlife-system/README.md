# SKYMP Death & Afterlife System - Enhanced

Full immersive death and afterlife mechanics for player-only RP servers.

## New Features Added
- **Haunting Mechanics**: Ghosts can haunt living players, causing notifications and potential RP effects.
- **Soul Gem Integration**: Living players can trap ghost souls in soul gems (persistent binding).
- **Visual Ghost Effects**: Translucent ghost appearance with visual shaders/effects on client.

## Original Features
- On death: Teleport to custom Afterlife realm
- Player-to-player interactions in afterlife (pacts, negotiations)
- Revival rituals by living players
- Configurable permadeath

## Installation
1. `npm install && npm run build`
2. Copy `Platform/` to SKYMP plugins
3. Add server logic to gamemode
4. Create custom Afterlife cell in CK

Press **F12** in afterlife. Use haunting and soul gems for deep RP!

Great for combining with Bounty and Territory systems.
