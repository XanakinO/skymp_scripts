# SKYMP Territory Control & Player Warfare System

## Features
- Players/guilds claim territories on the map
- Control grants resource nodes, fast travel (player-built), tax collection from players in area
- **Full Siege Mechanics**: Start sieges, join as attacker/defender, timed battles (30 min default)
- Victory determined by participant numbers (attackers vs defenders)
- Contested status with real-time sync and notifications
- Dynamic map markers for controlled/contested zones
- Persistent ownership and siege state across server restarts
- Ready for integration with Faction & Bounty systems

## Installation
1. Build the client plugin
2. Add server logic to your gamemode
3. Place Platform/ folder in SKYMP client plugins

Press **F11** to open Territory Management UI.

Built for player-only servers.
