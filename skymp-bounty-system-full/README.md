# SKYMP Bounty System - Player Crime & Bounty Hunting Framework

Fully player-driven bounty and crime system for **no-NPC** SKYMP servers.

## Features
- Players place bounties on other players
- Persistent wanted status
- Bounty Board UI (F10)
- Automatic rewards on player kills
- Server-authoritative (anti-abuse)
- Integrates with faction/reputation systems

## Installation
1. Copy `Platform/` folder to your SKYMP client plugins
2. Add `server/bounty.ts` logic to your server gamemode
3. Build with `npm install && npm run build`
4. Restart server & client

Press **F10** in-game to open the Bounty Board.
