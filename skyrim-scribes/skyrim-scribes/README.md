# Skyrim Scribes - High Fantasy Literature Addon for SkyMP

## Features
- Rich WYSIWYG book editor with Quill.js
- Realistic high-fantasy leather + parchment UI
- Dynamic Book Framework integration
- Local drafts + version history
- In-game Library Catalog with search & sharing

## Installation
1. Build with `npm install && npm run build`
2. Copy `dist/index.js` and `ui/` folder to `Data/Platform/Plugins/skyrim-scribes/`
3. Install Dynamic Book Framework
4. Create custom "Scribe's Journal" book in Creation Kit

See GitHub SkyMP docs for more.
