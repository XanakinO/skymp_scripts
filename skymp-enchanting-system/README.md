# SKYMP Custom Enchanting & Artifact Crafting System

## Features
- Player-crafted unique persistent artifacts
- Soul gem integration for powering enchantments
- Ritual Ceremony mechanics for powerful enchants
- Browser-based Enchanting UI (F9)
- Server-authoritative crafting with skill checks
- Fully persistent across sessions
- Perfect for player-only RP servers

## Installation
1. `npm install`
2. `npm run build`
3. Copy `Platform/` contents to your SKYMP client plugins folder
4. Integrate `server/enchanting.ts` into your gamemode or modules
5. Restart server and client

Press **F9** in-game to open the Arcane Forge UI.

**New: Ritual Ceremonies** – Gather participants for group rituals to create legendary artifacts with enhanced power.
