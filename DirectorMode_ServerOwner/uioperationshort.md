# Director UI Operation (short)

- Open the director UI: `/zeus` (or configured alias/keybind)
- Left column: pick **Actors / Items / Buildings**
- Center column: search + select entity + set **count** + click **SPAWN**
- Right column: tweak offsets (height/forward/right), waypoint/path controls, cleanup
- If other players don’t see changes: you need the server bridge to broadcast director events
